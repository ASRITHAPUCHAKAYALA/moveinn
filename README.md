# MoveInn – Caravan Booking & Travel Platform

##  Description

MoveInn is a web-based platform that combines **accommodation booking and ride services** through a caravan concept.
Users can select caravans based on their requirements and travel packages, enabling flexible and personalized travel experiences.

---

##  Features

* Caravan selection based on user requirements
* Package-based booking system
* Integrated accommodation + travel services
* User authentication (Login/Signup)
* Booking management and user dashboard
* Responsive and user-friendly interface

---

##  Tech Stack

**Frontend:**

* React.js
* JavaScript

**Backend:**

* Node.js

**Database:**

* MongoDB

## Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/ASRITHAPUCHAKAYALA/moveinn.git
```

---

### 2. Setup Frontend

```
cd moveinn-frontend
npm install
npm run dev
```

---

### 3. Setup Backend

```
cd moveinn-backend
npm install
npx nodemon server.js
```

---

##  Environment Variables

Create a `.env` file inside the backend folder and add:

```
MONGO_URI=your_mongodb_connection_string
```
